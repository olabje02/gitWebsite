Hvordan å git gud:

Steg 1: Initialiser en git repositorie i din mappe ved å bruke "git init".

Valgfri steg A: Legg til ett fjern repositorie ved å bruke "git remote add 'navn' 'url'".

Steg 2: Gjør tinger i repositorie.

Steg 3: Bruk "git add ." for å legge til alle filer eller "git add 'filnavn'" for å legge til filer til git repositoriet.

Steg 4: Bruk "git commit" for å large endringer i et commit, det er foreslått å bruke "git commit -m 'melding'" til å large din commit med en melding om hva som ble gjort.

Valgfri steg B: Sjekk etter fjern-repositorie endringer med "git fetch".

Valgfri steg C: Hvis det er endringer, bruk "git pull".

Valgfri steg D: Send dine endringer til fjern-repositorier med "git push".


Viktige git kommandoer:

Clone: Kloner ett repositorie til en ny mappe.

Init: Lager et nytt git repositorie i en mappe.

Add: Legger til filer til git indeksen.

Mv: Flytter eller git nytt navn til en fil eller mappe.

Restore: Restorerer arbeids tre filer

Rm: Fjerner filer fra arbeids treet og indeksen.

Bisect: Kjør ett binært søk etter en commit som skapte en bug.

Diff: Viser forskjellene mellom arbeids trær, committer, ogsv.

Log: Viser en logg over committer.

Status: Viser statusen av arbeids treet.

Fetch: Laster ned objekter og referenser fra en annen repositorie.

Pull: Laster ned og integrerer med en annen repositorie eller en lokal branch.

Push: Oppdaterer fjerne repositorier.
